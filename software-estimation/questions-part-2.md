1. Which estimation technique involves breaking down the project into small, well-defined components and estimating each one individually?

a) Delphi Technique
b) Top-Down Estimation
c) Bottom-Up Estimation
d) Expert judgment

Answer: c) Bottom-Up Estimation

2. What is the primary advantage of using the Delphi Technique for estimation?

a) speed and efficiency
b) high accuracy
c) collaboration among experts
d) reduces bias and groupthink

Answer: reduces bias and groupthink

3. In parametric modeling, what are the key variables used to estimate project effort and duration?

a) Expert opinions
b) Mathematical relationships based on historical data
c) gut feelings
d) market trends

Answer: Mathematical relationships based on historical data

4. What is the primary purpose of function point analysis in software estimation?

a) To estimate project duration
b) To measure the functionality provided by software
c) To estimate the number of lines of code
d) To predict resource availability

Answer: b) To measure the functionality provided by software

5. Which estimation technique is based on the idea that the cost of a project is proportional to its size, often measured in lines of code or function points?

a) Expert Judgment
b) Lines of Code (LOC) Estimation
c) Delphi Technique
d) Parametric Modeling

Answer: d) Parametric Modeling

6. What is the primary focus of COCOMO (COnstructive COst MOdel) in software estimation?

a) Function points analysis
b) Expert opinions
c) Risk management
d) Predicting project effort based on project size

Answer: d) Predicting project effort based on project size

7. Which estimation technique often relies on historical data and uses analogy to estimate a new project's effort and duration?

a) Expert Judgment
b) Lines of Code (LOC) Estimation
c) Analogy-Based Estimation
d) Parametric Modeling

Answer: Analogy-Based Estimation

8. What is a key challenge associated with the use of parametric estimation models?

a) They are time-consuming to use
b) They do not consider project size
c) They require a significant amount of historical data
d) They do not account for expert opinions

Answer: c) They require a significant amount of historical data

9. What is the main benefit of using expert judgment in software estimation?

a) High accuracy
b) Speed and efficiency
c) Objectivity
d) Valuable insights from experienced professionals

Answer: d) Valuable insights from experienced professionals

10. Which estimation technique is particularly useful when there is a lack of historical data or detailed project information?

a) Expert Judgment
b) Lines of Code (LOC) Estimation
c) Function point analysis
d) Parametric Modeling
